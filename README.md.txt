# 🏥 Dashboard Saúde Saquarema

Dashboard interativo para acompanhamento de metas de saúde do município de Saquarema.

## 📊 Funcionalidades

- **Monitoramento em Tempo Real**: Acompanhamento proporcional baseado na data de extração
- **Status por Cores**: 
  - 🟢 Verde: No prazo
  - 🔵 Azul: Acima do esperado  
  - 🔴 Vermelho: Abaixo do esperado
- **Gráficos Interativos**: Donut, barras e tabelas dinâmicas
- **Filtros Personalizáveis**: Por unidade e indicador
- **Download de Dados**: Exportação em CSV

## 🎯 Indicadores Monitorados

- **Consulta de Enfermagem** (Meta: 240/mês)
- **Consulta Médica** (Meta: 300/mês)  
- **Atendimento Odontológico** (Meta: 200/mês)
